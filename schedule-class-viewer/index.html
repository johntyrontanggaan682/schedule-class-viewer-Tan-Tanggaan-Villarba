About

System Purpose

The SLSU Lucena BIT Class Schedule Viewer provides a centralized, real-time way to view academic schedules for the BIT program at Southern Luzon State University Lucena Campus. It focuses on clear, student and faculty-friendly viewing of class grids, sections, and assignments to minimize conflicts and make information easy to find.

Key Features

Interactive Viewing

Weekly grid with subjects, rooms, teachers, and edit actions.
